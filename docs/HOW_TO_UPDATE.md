# How to Update Website Daily

Open `admin.html`.

Edit the required section:
- academy
- contact
- notices
- infrastructure
- courses
- faculty
- facultyPosters
- fees
- admission
- gallery

Then click **Download Updated site-data.js** and replace:

`data/site-data.js`

## Images
For faculty photos use:

`assets/images/faculty/`

For faculty posters use:

`assets/images/faculty-posters/`

Then edit paths in `admin.html`.

## Note
The previous RAR archive was very large. This updated website includes generated faculty introduction posters and available photos/posters from the current working files. Real missing photos can be added later by replacing placeholder images.
